## 📌 About the Project

Atiqa’s Digital World is a personal portfolio ecosystem that brings together multiple web projects under one unified digital identity.

It is designed to represent:
- A growing frontend development journey  
- Experimentation with UI/UX concepts  
- Brand-level presentation of projects  
- A structured showcase of deployed websites  

Each project inside reflects a different theme, design system, and user experience approach.

- 🧭 Centralized portfolio hub for multiple projects

- 🔗 Direct access to live deployed websites

---

## 🧠 Design Thinking

The platform is structured around usability and clarity:

- **Affordance:** Project cards clearly indicate clickability and navigation  
- **Constraints:** Minimal clutter to keep focus on projects  
- **Mapping:** Logical grouping of projects for easy exploration  
- **Causality:** Immediate navigation feedback when interacting with project links  

The goal was to create a **developer portfolio that feels like a digital exhibition space** rather than a static resume page.
